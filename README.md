# FER-2013

A deep learning architecture for classifying facial emotions, learning from FER-2013 dataset.

## Project Structure

```bash
fer-2013
├── configs
├── data
├── environment.yml
├── LICENSE
├── notebooks
├── README.md
├── src
└── tests
```

## Setup

```bash
conda env create -f environment.yml
conda activate fer-2013
```
